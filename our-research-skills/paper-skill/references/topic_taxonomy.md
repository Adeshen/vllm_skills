# Topic Taxonomy

Use these topic labels when classifying papers.

| Topic | Description | Local destination |
| --- | --- | --- |
| `inference_engine_bugs` | Empirical bug studies for LLM inference engines. | bug wiki capsules, workload patterns |
| `serving_fuzzing` | Fuzzing or active testing of serving systems. | fuzzer playbooks, GRIEF seed design |
| `incident_management` | Production incidents, TSGs, SRE automation. | incident triage, report skills |
| `workflow_security` | GitHub Actions, agentic workflow injection, taint analysis. | skill safety, CI automation hardening |
| `multi_arch_ci` | Cross-architecture or backend-specific CI strategy. | CPU/CUDA/NPU matrix design |
| `serving_architecture` | Scheduler, batching, KV cache, disaggregation, serving system design. | feature wiki deep dives |
| `multimodal_diffusion_serving` | Stage-level diffusion or multimodal serving. | feature wiki stage/disaggregation pages |
| `kernel_optimization_agents` | Agentic search, kernel generation, profiling, optimization. | HPC/MLSys skills, operator optimization |
| `evaluation_quality_gate` | Evaluation harnesses, quality gates, benchmark methodology. | verification experiments |
