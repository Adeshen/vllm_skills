# Skill Discovery References

This directory contains Git submodules that help discover, rank, evaluate, or govern large skill catalogs.

| Source | Path | Best use |
| --- | --- | --- |
| Skills Vote | [skills-vote/](skills-vote/) | Large-scale skill discovery, evaluation, recommendation, and governance workflow reference. |

## Local Use

Use this as a method reference when expanding the local skill repository. Do not import discovered skills blindly; add them as submodules or local wrappers only after checking scope, license, schema compatibility, and relevance to vLLM, HPC, MLSys, or the active research corpus.
